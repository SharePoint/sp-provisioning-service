SharePoint provisioning service

Provisioning is currently disabled due to a maintenance break.

We expect the service to be online again in January 2022.

If needed, you can use PnP PowerShell to provision the templates manually. All templates that are visible in the service are available from GitHub repository, which acts as the source location for all the visible information in this service. You can download the used templates and related assets directly from GitHub and then use PnP PowerShell to use them also outside of this service.
